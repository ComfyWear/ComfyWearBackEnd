# ComfyWearBackEnd

For ged:
```
git clone --filter=lfs https://github.com/ComfyWear/ComfyWearBackEnd.git
cd ComfyWearBackEnd
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
